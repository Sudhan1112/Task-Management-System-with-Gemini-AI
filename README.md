# Task Management System with Gemini AI

## Overview
A task management application that allows users to manage tasks via a React UI or natural language commands processed by Gemini AI.

## Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Python, Django, Django Rest Framework
- **Database**: PostgreSQL
- **AI**: Google Gemini API
